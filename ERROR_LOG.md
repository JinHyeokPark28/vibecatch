> **처리 방법** - `/learn-error` 실행 → 패턴 분석 → CLAUDE.md 규칙화 → 이 파일 비움

---

## 에러 기록 규칙

### Claude가 자동으로 기록할 때
- 모든 에러 발생 시 (빌드, 테스트, 런타임)
- 같은 에러는 횟수만 +1
- 해결되면 상태 업데이트

### 기록 형식
```markdown
### [에러-시그니처]
- **횟수**: N
- **최근**: YYYY-MM-DD HH:MM
- **위치**: 파일:라인
- **입력**: 관련 입력값
- **에러**: 에러 메시지
- **상태**: ❌ 미해결 / ✅ 해결
- **메모**: 추가 정보
```

### 에스컬레이션
- **3회 반복** → 🚨 자동 `/deep-debug` 트리거
