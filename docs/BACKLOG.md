## 대기 중

### B001: 액션 가능한 인사이트 요약

**설명**: 현재 짧은 요약을 확장하여 실질적으로 활용 가능한 인사이트 제공

**요청 배경**:
- 현재 3줄 요약으로는 상세 내용 파악 어려움
- 단순 정보 전달이 아닌 "어떻게 활용할지"가 중요
- 바이브 코더에게 실질적 가치 제공

**출력 포맷**:
```
📌 핵심 요약
• 포인트 1
• 포인트 2
• 포인트 3

💡 개선안
• 이 기술/아이디어의 개선점
• 한계점 극복 방안

🛠️ 적용 방법
• 내 프로젝트에 어떻게 적용할지
• 구체적인 액션 아이템들
```

**예상 구현**:
1. summarizer.py 프롬프트 수정
2. 출력 포맷 변경 (summary → insight 구조화)
3. DB 스키마는 기존 summary 컬럼 재활용 가능

**예상 비용**: 기존과 비슷 (토큰 약간 증가)

**기술적 고려사항**:
- 프롬프트 엔지니어링 필요
- JSON 구조화 출력 파싱
- 기존 데이터 마이그레이션 불필요 (새 수집분부터 적용)

**우선순위**: 중간 (MVP 완료, 핵심 가치 향상)

**상태**: 대기
